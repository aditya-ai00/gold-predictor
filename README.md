# 🪙 GoldVision AI — Elite Gold Market Terminal

GoldVision AI is a production-ready, minimalist full-stack quantitative terminal designed for analyzing gold spot valuation metrics ($XAU/USD$), Indian domestic retail precious metal physical markets, and broader macroeconomic indicators. 

Powered by **Next.js 15 (App Router)** and the **Google Gemini 2.5 Engine**, it delivers automated prediction modelling, mathematical jewelry margin tools, portfolio yield tracking, and a live conversational AI assistant.

---

## 🚀 Core Features

- **Live Market Station:** Real-time spot price tracking with an algorithmic asset evaluation curve using Recharts.
- **Neural AI Prediction:** On-demand generation of comprehensive directional targets (Bullish/Bearish) and short, weekly, and monthly macro outlooks via the Gemini API proxy.
- **Premium Jewelry Calculator:** Complete computational breakdown of fine metal gram weight pricing across 22K and 24K pure benchmarks, incorporating maker premiums and statutory standard GST metrics.
- **Asset Ledger Matrix:** Real-time tracking of physical bullion configurations, volume yields, acquisition history, and historical profit performance.
- **Cognitive Terminal Interface:** Direct chat stream targeting precious metals questions, financial observations, and market volatility updates.

---

## 🛠️ Technology Stack

- **Framework:** Next.js 15 (App Router under Turbopack compilation)
- **Frontend & Rendering:** React 19, TypeScript
- **Styling Core:** Tailwind CSS (v4 Architecture via structural `@import`)
- **Charts:** Recharts Quantitative Component Framework
- **Icons:** Lucide React
- **AI Core:** Google Gen AI SDK (`@google/genai`)
